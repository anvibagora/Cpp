Setw
    setw(value) ----> set width; the vaule is the number of characters it takes.
    It is used to format the output.
    eg: let a=1 and b=1234
        cout<<"Without setw:"<<a<<endl;
        cout<<"Without setw:"<<b<<endl;
        cout<<"with setw:"<<setw(4)<<a<<endl;
        cout<<"with setw:"<<setw(4)<<b<<endl;
    output:
        Without setw:1
        Without setw:1234
        With setw:   1
        with setw:1234

Structure
    User defined data type. Derived data type. Is made of multiple data types. All variables should have values.


Union
    User defined, derived data type. Is made up of multiple data type. Only one variable can be used at a time.
    Union saves more memory space than structure.----->In structure all variables will have a value but in
        union only one variable will have a value.
